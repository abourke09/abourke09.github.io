Mad Hippie

CLI Data Gem Portfolio Project

Blog Post
GitHub
YouTube

Mad Hippie is a natural skincare company offering products for sale through their website. This CLI gem will help you to learn more about their product line by listing each item and making the product details available to view.

  • Written in object-oriented Ruby to accept user input directly from the command line
  • Scrapped the Mad Hippie website using Nokogiri to collect product information
