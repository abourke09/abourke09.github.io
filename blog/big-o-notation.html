Big O Notation

Understanding and improving upon your code's efficiency

Computer scientists use Big O notation to describe how efficient their code is. Time efficiency and space efficiency are both a part of this, although it's more common to focus on the former (runtime). It is of course more complicated than that, but that’s the core concept. Big O notation is important to us as software engineers and web developers because knowing how to measure code’s efficiency is the first step to improving it. Obviously, the less efficient the code is and the longer the runtime, the longer the delay is for the user.

It’s also pretty common for Big O notation to be brought up in an interview setting. Even if you’ve written a function that works perfectly, your interviewer may ask you to make an adjustment to improve upon its runtime. Knowing how to measure, communicate about, and improve your function’s efficiency is important.

So what’s the best possible runtime and what’s the worst? And how can we talk about it in the context of Big O? The graphic below does a great job of ranking runtimes according to emojis. It was created by Starr Horne for Honeybadger and is available here.

Each of the functions below serves as an example of a different runtime.

For a function of O(1), the runtime is constant and does not depend on the input at all.

									
input = ["cat", "dog", "horse"]

def my_function(input)
	puts "Hello #{input[1]}"
end

#Output:
#Hello cat

For a function of O(n), the runtime is linear and only runs through the input data once.

									
input = ["cat", "dog", "horse"]

def my_function(input)
	input.each do |i|
		puts "Hello #{i}"
	end
end

#Output:
#Hello cat
#Hello dog
#Hello horse

For a function of O(n^2), the runtime is squared and tends to involve nested loops.

									
input = ["cat", "dog", "horse"]

def my_function(input)
  input.each do |first|
    input.each do |second|
      puts "Hello #{first}, said the #{second}."
    end
  end
end

#Output:
#Hello cat, said the cat.
#Hello cat, said the dog.
#Hello cat, said the horse.
#Hello dog, said the cat.
#Hello dog, said the dog.
#Hello dog, said the horse.
#Hello horse, said the cat.
#Hello horse, said the dog.
#Hello horse, said the horse.
